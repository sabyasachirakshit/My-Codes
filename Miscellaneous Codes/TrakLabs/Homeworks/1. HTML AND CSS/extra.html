HTML AND CSS LIBRARY

HTML

HyperText Markup Language (HTML) is a type of markup language. It is used to make webpages. Webpages can include writing, links, pictures, and even sound and video. HTML tells web browsers what webpages should look like.

CSS

Cascading Style Sheets, fondly referred to as CSS, is a simple design language intended to simplify the process of making web pages presentable. CSS is used to control the style of a web document in a simple and easy way.
